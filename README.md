# Predicting density and assessing factors driving marbled murrelet abundance and distribution in Puget Sound using hierarchical distance sampling (2026)

Gillman, S.J., Pearson, S.F., Lance, M.M., Duarte, A., Gardner, B. 

R code and data submitted for review to the journal *Ecology and Evolution*
---

## Repository Directory

- **[CovariateProcessing](https://github.com/sjgillman/PS_MAMU/tree/main/CovariateProcessing)**: Contains code for creating grid cells, aggregating static and dynamic covariates
- **[DataAnalysis](https://github.com/sjgillman/PS_MAMU/tree/main/DataAnalysis)**: Contains code for formatting observation data, modeling, analysis, and results.
- **[PostProcessing](https://github.com/sjgillman/PS_MAMU/tree/main/PostProcessing)**: Contains code to estimate posterior probabilities and create figures.
